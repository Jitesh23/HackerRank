# Day 1: Quartiles

### Output Format

Print 3 lines of output in the following order:

1. The first line should be the value of Q¹.
2. The second line should be the value of Q².
3. The third line should be the value of Q³.

### Sample Input

```
9
3 7 8 5 12 14 21 13 18
```

### Sample Output

```
6
12
16
```
